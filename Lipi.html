All-In-One Collaboration and Productivity Platform

A Note Taking app built using Next.js (App Router), Typescipt, Tailwind CSS, shadcn/ui, Drizzle ORM & more!

Get StartedGithub

Tech Stack

This project is an experiment to see how a modern app, with features like auth, subscriptions, API routes, and static pages would work in Next.js app dir.

Next.js

App dir, Routing, Layouts, Loading UI and API routes.

React 18

Server and Client Components. Use hook.

Database

ORM using Drizzle and deployed on Subabase.

Components

UI components built using shadcn/ui and styled with Tailwind CSS.

Authentication

Authentication using NextAuth.js and middlewares.

Subscriptions

Free and paid subscriptions using Stripe.

Keep track of your meetings all in one place

Capture your ideas, thoughts, and meeting notes in a structured and organized manner.
